1. @ConfigurationProperties adalah sebuah bind kepada suatu object java, yaml berisikan konfigurasi dan setting untuk aplikasi berjalan, di bind kepada object
2. ResourceLoader adalah sebuah interface abstrak untuk kegunaan mengambil data seperti text, gambar, atau lainnya
3. MessageSource tujuannya adalah untuk mengamil text berdasarkan kode bahasa seperti en atau id, membuat aplikasi menjadi dinamis untuk mengubah language yang digunakan user
